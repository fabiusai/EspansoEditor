💡 Guida alle Variabili di Espanso

Espanso supporta l'inserimento di contenuti dinamici chiamati "Variabili". Ecco le sintassi dei formati intelligenti più utili.

1. Posizionamento del Cursore

Usa la stringa $|$ all'interno del tuo testo. Espanso incollerà tutto e poi posizionerà il cursore lampeggiante esattamente in quel punto, pronto per farti scrivere.

Gentile $|$, ti scrivo in merito alla pratica...

2. Data e Ora Corrente

Per inserire la data di oggi in modo automatico, usa la variabile {{mydate}} nel testo e definisci il formato nel blocco vars.

replace: "Oggi è il {{mydate}}" vars: - name: mydate type: date params: format: "%d/%m/%Y"

3. Comandi di Sistema (Shell)

Utile se vuoi eseguire un comando dal terminale e incollare l'output direttamente nel testo.

replace: "Il mio IP locale è: {{ip}}" vars: - name: ip type: shell params: cmd: "ipconfig getifaddr en0"

4. Incolla Dagli Appunti (Clipboard)

Permette di prendere il testo che hai attualmente copiato negli appunti e avvolgerlo in una frase standard.

replace: "Codice di tracciamento copiato: {{clipboard}}" vars: - name: clipboard type: clipboard

Nota di utilizzo: Attualmente questo editor visuale permette di modificare comodamente il "Trigger" e il testo base ("Replace"). Se desideri aggiungere o modificare le configurazioni complesse delle variabili (il blocco `vars:`), puoi scriverle manualmente nel file base.yml e re-importarlo qui. L'app le conserverà in memoria in modo sicuro senza cancellarle!
